# 🔄 Converter

Application web de conversion d'unités développée en HTML, CSS et JavaScript pur.

## 📋 Présentation

Converter est un outil simple et intuitif permettant de convertir des valeurs entre différentes unités de mesure directement depuis le navigateur, sans aucune dépendance externe.

## ✨ Fonctionnalités

- Conversion de longueurs (km, m, cm, mm, miles, pieds, pouces)
- Conversion de masses (kg, g, mg, tonnes, livres, onces)
- Conversion de températures (Celsius, Fahrenheit, Kelvin)
- Résultat instantané à la saisie
- Interface responsive, utilisable sur mobile et desktop

## 🛠️ Technologies utilisées

| Technologie | Usage |
|---|---|
| HTML5 | Structure de l'interface |
| CSS3 | Mise en forme et responsive design |
| JavaScript (ES6+) | Logique de conversion et manipulation du DOM |

## 🚀 Lancer le projet

Aucune installation requise. Il suffit de cloner le dépôt et d'ouvrir le fichier `index.html` dans un navigateur.

```bash
git clone https://github.com/PecqueurFlorian/converter.git
cd converter
```

Puis ouvrir `index.html` dans votre navigateur.
